# Electronic Clock
    ------Based On C51

#### `项目介绍`
-----------------------------------------------------------------------------------------------

    本项目是基于STC51的电子时钟设计，完成以下功能：
    
      ·时钟计时及显示
        
      ·温度测量及显示
